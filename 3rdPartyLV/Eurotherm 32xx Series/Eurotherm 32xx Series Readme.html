Eurotherm 32xx Series Instrument Driver Readme

To rate this instrument driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Eurotherm
Supported Language(s): LabVIEW
Supported Model(s): 3204, 3208, 3216, 32H8, 32H8i, 3216i
Model(s) Tested: 3204, 3208, 3216
Interface(s): Serial
Firmware Revision(s) Tested: v213, v232

Certified: Yes
NI Supported: Yes
Source Code Available: Yes

Driver Revision: 1.0
Original Release Date: 02/24/2010
Current Revision Date: 02/24/2010

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 3.0 or later

Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

3. Known Issues

The 32xx Range Instruments conform to MODBUS RTU ® protocol and do not support Error Query. Various Diagnostic alarms are used to indicate a possible fault within the controller or connected devices. For further information refer to Instrument Manual.

To report issues or provide feedback about this instrument driver, please send an email to instrument.drivers@ni.com.
